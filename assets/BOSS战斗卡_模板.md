# BOSS 战斗卡 · 模板

> 服务对象：战斗策划
> 生成原则：机制可追溯索引；阶段对应原文情节；**禁止给出具体数值**

---

## 卡字段定义

| 字段 | 必填 | 说明 | 来源 | 备注 |
|------|:----:|------|------|------|
| BOSS名 | ✓ | 与 MO 索引对应 | MO 索引 | — |
| 建议等级段 | △ | 区间范围（如 30-40 级） | 人工判断精彩度/战斗强度 | 可省略 |
| 战斗形象 | ✓ | 战斗阶段的外观变化 | 原文战斗/变身描写 | 原文无则用"原始形态" |
| 技能包 | ✓ | 3-5 个技能，每个映射 SP-ID | SP 索引 | 必须可回溯 |
| 场景机制 | △ | 基于 PL/RI 索引的地形/仪式机制 | PL/RI 索引 | 无关联则省略 |
| 阶段设计 | ✓ | 2-3 阶段，对应原文情节转折 | 原文战斗进程 | 必须对应真实情节 |
| 掉落参考 | △ | 基于 IT 索引的故事相关道具 | IT 索引 | 无关联则省略 |
| 结局挂钩 | △ | 多结局/分支对应 | 原文结局 + 索引游戏建议 | 无分支则省略 |
| **数值字段** | ✗ | 禁止给出具体数值 | — | HP/DMG/CD 全部禁用 |

## 标准输出模板

```
═══════════════════════════════
【BOSS】{BOSS名} · 建议 {X-Y} 级
═══════════════════════════════
◆ 战斗形象：
   {战斗阶段的外观描述，包括变身/揭露等}

◆ 技能包（源自 SP 索引）：
   · {技能名}（{SP-ID}）→ {定性描述：范围/单体/控制/DOT/位移等}
   · {技能名}（{SP-ID}）→ {定性描述}
   · {技能名}（{SP-ID}）→ {定性描述}

◆ 场景机制（源自 PL/RI 索引）：
   · {场所名}（{PL-ID}）→ {地形效应}
   · {仪式/桥段名}（{RI-ID 或 BR-ID}）→ {阶段转换触发点}

◆ 阶段设计：
   · 一阶段「{原文情节对应}」：{主要技能+行为模式}
   · 二阶段「{原文情节对应}」：{转换触发+新技能}
   · 三阶段（如有）「{原文情节对应}」：{终局行为}

◆ 掉落参考（源自 IT 索引）：
   · {道具名}（{IT-ID}）{如有副作用则标注}

◆ 多结局挂钩：
   · {结局分支1：对应原文结局}
   · {结局分支2：对应索引游戏建议}

◆ 来源：{MO-ID} / 故事 {TC-ID 或 YYCZ-ID}
═══════════════════════════════
```

## 合格示例

```
═══════════════════════════════
【BOSS】板桥三娘子 · 建议 30-40 级
═══════════════════════════════
◆ 战斗形象：
   一阶段：人形驿店老板娘（伪装）
   二阶段：施法现原形，魔饼悬浮周身

◆ 技能包（源自 SP 索引）：
   · 食物变形术（SP-007）→ 范围异常状态（中招者变驴倒计时）
   · 魔饼投掷（原创，基于 IT-023）→ 单体远程+持续异常
   · 驿店召唤（基于 PL-031）→ 召唤已变驴的旧旅客协战

◆ 场景机制（源自 PL 索引）：
   · 板桥驿店（PL-031）→ 封闭地形，无法撤离直至 BOSS 倒下或认输
   · 厨房区域 → 打破灶台可中断 BOSS 的"变形术"充能

◆ 阶段设计：
   · 一阶段「伪装待客」：以对话/任务形式引玩家卸防，首次出手突袭
   · 二阶段「真身揭露」：被识破后转入战斗，主用变形术和召唤
   · 三阶段「反噬触发」：被逼急后自身开始变驴，攻击模式紊乱

◆ 掉落参考（源自 IT 索引）：
   · 魔饼（IT-023）带副作用：携带过久施法者自身会缓慢变形

◆ 多结局挂钩：
   · 击杀：驿店解封，所有变驴旅客无法复原
   · 制服迫其解术：所有变驴旅客恢复人身（原文"反噬"设定延伸）
   · 逃脱：驿店变为长期危险点，未来可能再触发

◆ 来源：MO-068 / TC-068《板桥三娘子》
═══════════════════════════════
```

## 常见违规警示

### 违规 1：给出具体数值
```
❌ 一阶段 HP 120000 / 魔饼 DMG 3500 / CD 8s
```
理由：数值是战斗策划的领地，本卡只给定性描述。

### 违规 2：编造原文没有的技能
```
❌ 技能包包含"暗影冲刺 → 位移类"（原文无任何描述）
```
理由：技能必须能映射回 SP 索引或原文明确描述，不得创造。

### 违规 3：阶段切换脱离原文
```
❌ 二阶段「狂暴化：攻击力+50%」
```
理由：原文中并无"狂暴"情节，阶段切换必须对应原文情节转折（如揭露/反噬/召唤），不得套用通用游戏模板。

3. 对照 `./卡生成质检清单.md` 逐项勾选
