# 妖怪/精怪角色卡 · 模板

> 服务对象：叙事策划
> 生成原则：忠于原文，严禁脑补；三要素齐全；叙事钩子要具体

---

## 卡字段定义

| 字段 | 必填 | 说明 | 来源 | 留白规则 |
|------|:----:|------|------|---------|
| ID | ✓ | MO-编号 | MO 索引 | — |
| 名称 | ✓ | 妖怪正式名 | MO 索引 | — |
| 精彩度 | ✓ | ⭐ / ⭐⭐ / ⭐⭐⭐ | MO 索引 | — |
| 一句话定位 | ✓ | ≤20字，说清"物种+关键特征" | MO 简述 | — |
| 形象档案 | △ | 形态/性别/服饰/标志性外观 | 原文描述 | 原文无则留"原文未述" |
| 能力标签 | ✓ | 2-4 个，关键词形式 | MO 核心能力字段 | — |
| 叙事钩子 | ✓ | 2-4 条，**必须是具体冲突/反差**，禁评价词 | 原文情节+角色档案 | — |
| 配套资源包 | ✓ | SP/IT/PL/RI/MT 关联 | 各索引交叉查 | 未命中则某类省略 |
| NPC交互模板 | △ | 性格三词+NPC型+玩家交互分支 | 原文行为倒推 | 原文只做背景未交互→此字段省略 |
| 叙事用法建议 | ✓ | BOSS型/NPC型/背景型分类给出 | MO 游戏设计建议原文 | — |
| 出处 | ✓ | 故事 ID + 作品名 | — | — |

## 标准输出模板

```
═══════════════════════════════
【{ID}】{名称} · {精彩度}
═══════════════════════════════
◆ 一句话定位：{≤20字，物种+关键特征}

◆ 形象档案：
   {形态/性别/服饰/标志性外观，原文无则标"原文未述"}

◆ 能力标签：
   {标签1} ｜ {标签2} ｜ {标签3}

◆ 核心叙事钩子：
   · {具体冲突/反差描述1}
   · {具体冲突/反差描述2}
   · {具体冲突/反差描述3}

◆ 配套资源包：
   · 法术：{SP-ID 名称}
   · 道具：{IT-ID 名称}
   · 场所：{PL-ID 名称}
   · 仪式：{RI-ID 名称}
   · 母题：{MT类+关键词}

◆ NPC交互模板：
   【性格三词】{词1} / {词2} / {词3}
   【做成{X}型NPC】
   【玩家交互】{交互分支，建议给 2-3 种}

◆ 叙事用法建议：
   · BOSS型：{具体场景}
   · NPC型：{具体场景}
   · 背景型：{具体场景}
   （未命中的类型省略）

◆ 出处：{故事ID}《{作品名}》
═══════════════════════════════
```

## 合格示例

```
═══════════════════════════════
【MO-068】板桥三娘子 · ⭐⭐⭐
═══════════════════════════════
◆ 一句话定位：板桥驿店女主人，以魔饼化客为驴

◆ 形象档案：
   三十余岁独居妇人，经营驿店，表面热情好客
   （原文未提供具体外貌细节）

◆ 能力标签：
   变形术 ｜ 食物下蛊 ｜ 伪装经营

◆ 核心叙事钩子：
   · 驿店里每一头驴都曾是人——旅客以驴的身份被卖掉
   · 吃下一顿看似寻常的饼，意味着彻底失去人身
   · 施术者最终也无法逃脱同样命运（反噬循环）

◆ 配套资源包：
   · 法术：SP-007 食物变形术
   · 道具：IT-023 魔饼
   · 场所：PL-031 板桥驿店
   · 仪式：—
   · 母题：C 变形/变身 + H 禁忌秘密

◆ NPC交互模板：
   【性格三词】热情 / 隐忍 / 残忍
   【做成伪装型危险NPC】
   【玩家交互】
     ① 旅宿过夜触发剧情
     ② 察觉异常可侦查搜证
     ③ 发现真相后对峙或逃离

◆ 叙事用法建议：
   · BOSS型：揭露型战斗（发现真相后触发）
   · NPC型：都市传说源头（早期铺垫）
   · 背景型：驿店副本的核心谜题设计

◆ 出处：TC-068《板桥三娘子》（古代前代传奇集）
═══════════════════════════════
```

## 不合格示例（对比学习）

### 反例 1：脑补细节

```
◆ 形象档案：
   身着素色罗裙，鬓发斜簪玉钗，眉目之间隐有寒光，走路时
   衣袂无风自动……
```
❌ 错误：原文只有"独居妇人"，其余是AI脑补画面。
✅ 修正：如形象档案模板示例所示，原文无则写"原文未述"。

### 反例 2：叙事钩子空洞

```
◆ 核心叙事钩子：
   · 经典的变形术题材
   · 非常精彩的志怪故事
   · 值得一用的妖怪角色
```
❌ 错误：全是评价词，没有具体冲突。
✅ 修正：必须写清楚"冲突点在哪里/反差在哪里/悲剧性/恐怖点"的具体内容。

### 反例 3：未留白

```
◆ NPC交互模板：
   【性格三词】温柔 / 狡诈 / 深沉
   【做成诱导型NPC】
   【玩家交互】可对话、可攻略、可结盟……
```
❌ 错误：原文只描述妇人做生意、下蛊、变驴，并无"温柔""深沉"等情感侧写依据，且"可攻略"完全脱离原文。
✅ 修正：性格词必须能从原文行为倒推，无据则删除此字段。
