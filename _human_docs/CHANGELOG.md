# CHANGELOG

## v3.0（2026-05-06）

### 定位
- 志怪文学游戏叙事素材库（只是一个志怪库的定性，不绑定具体朝代）
- 唐传奇、酉阳杂俎等具体作品名在 Skill 输出中正常使用，不刻意避讳
- 面向未来纳入更多朝代/更多志怪作品，架构通用化

### 能力收敛
- **只保留「素材检索」核心能力**，新增「角色卡生成」增值能力
- 以下能力转为 `roadmap/` 待拓展：故事组装 / 母题推荐 / 桥段情绪筛选 / 作品评估

### 入口调整
- 策划使用入口从 10 个收敛至 4 个，编号按 ①②③④ 重新排列：按素材类型 / 按母题套路 / 按桥段功能 / 查神明角色
- 入口③（桥段功能）降级：仅按功能类/子类检索，不启用情绪标签筛选

### 数据变更
- **新增**：叙事母题索引独立编号（MT）、叙事桥段索引独立编号（BR）
- **删除**：社会身份角色索引，人物参考走神明库或全文层角色档案
- **桥段索引升级 v3**：编剧视角重构版，394 条，18 类细分（E 拆 7 子类 + D 拆 2 子类）
- **8 大索引实际规模**（以各索引文件真实统计为准）：
  - MO 妖怪精怪 312 条
  - SP 法术能力 361 条
  - IT 法器道具 730 条
  - PL 场所空间 128 条
  - RI 仪式民俗 414 条
  - GD 神明角色库 122 位（5 界别）
  - MT 叙事母题 13 大类（覆盖 156 篇唐传奇）
  - BR 叙事桥段 v3 394 条
  - **合计约 2,474 条**

### 新增能力
- **两种角色卡**：妖怪/精怪角色卡（叙事向）+ BOSS 战斗卡（战斗向）
- **五条硬约束**：忠实性 / 三要素完整性 / 反空洞 / 分层留白 / 游戏化克制
- **七项质检清单**：生成前自检，未通过禁止输出
- **预生成 5 张示范卡**，其余按需生成

### 文件结构
- 在原 `zhiguai-narrative-skill/` 下新建 `zhiguai-codex/` 子目录
- 采用 Claude Skills 官方推荐布局：`SKILL.md + assets/ + references/ + _human_docs/`
- 本版本稳定后，将取代整个旧 `zhiguai-narrative-skill/`
