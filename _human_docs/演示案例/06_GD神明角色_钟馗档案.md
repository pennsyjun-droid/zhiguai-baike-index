   · 二阶段「斩鬼」：正式战斗，主用剑术与符咒

## 本案例展示的关键能力

1. **GD 单体查询起点 L2 的规则兑现**：策划说"钟馗的完整资料"直接出六板块档案，不走 L1 列表
2. **六板块档案的完整形态**：基本 / 形象 / 特性 / 身世 / 关联 / 游戏价值，每一板块都有价值
3. **界别 + 功能二维筛选**：幽冥界 × 执法型的精确命中
4. **神明→BOSS 卡的转译**：神明库六板块的"游戏设计价值"字段直接驱动 BOSS 卡的三阶段设计
5. **正直度机制严守原典**：三阶段的"放水/觉醒"分支来自原文"嫉恶如仇"的选择性特征，不是为了游戏效果凭空造的机制
