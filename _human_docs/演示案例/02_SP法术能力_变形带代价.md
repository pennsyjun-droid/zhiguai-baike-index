## 本案例展示的关键能力

1. **字段级筛选价值**：同样是 SP 索引，加上"禁忌/代价"筛选后命中的才是有博弈空间的设计
2. **从法术反查妖怪**：SP → MO 的跨索引关联，很多策划只知道妖怪不知道法术，这个方向反向查也要能走通
3. **L1 + L2 同一条目的内容差**：L1 只能看到简述与代价摘要，L2 有完整机制描述
4. **自我克制的诚实声明**：最后一轮策划问"设计哲学"，AI 主动说明"本期不做哲学提炼，只做检索"，不越界
