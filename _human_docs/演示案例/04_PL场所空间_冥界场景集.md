## 本案例展示的关键能力

1. **界别批量查询**：一次拉出 PL 索引整个"幽冥界"分类
2. **功能排序**：按"入冥动线"这种真实使用场景重排，而非按 ID 排序
3. **跨索引全家桶**：一个副本需要的 MO/GD/RI/IT/MT/BR 全部串起来
4. **素材清单化交付**：最后以"核心 NPC / 核心道具 / 核心机制"形式结构化汇总，便于策划直接复制
