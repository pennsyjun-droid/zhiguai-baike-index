## v3.1：桥段情绪筛选启用

### 当前阻碍
桥段索引 429 条，情绪标签（35 标签 / 7 维度）尚未批量标注完成。

### 需要完成的前置工作
- 批量给 429 条桥段打情绪标签（E1-E7 七维度）
- 预估工作量：2-3 天专注标注

### 启用后效果
入口⑦ 从"按 10 类功能"升级为"按功能 × 情绪双维度检索"。

### 示例（启用后）
```
策划："来个让玩家细思极恐的真相揭露型桥段"
→ 查 BR 索引 B 类 × E1「细思极恐」
→ 命中：B056 崔慎思（妻子留下头颅，"忘哺乳"暗示）
```

## v3.2：母题关联推荐

### 当前阻碍
母题索引 287 条仅做了分类，**跨母题组合**未建立。

### 需要完成的前置工作
- 建立母题组合表（如 B跨界恋情 + H禁忌秘密 常在一起出现）
- 标注每种组合的"经典对应故事"

### 启用后效果
```
策划："我要做一个人鬼恋带复仇的故事"
→ 识别为 B跨界恋情 + A复仇报应
→ 推荐组合匹配度高的案例
→ 附带经典故事参考
```

## v3.3：故事组装引擎

### 当前阻碍
v3.1 情绪标签 + v3.2 母题组合必须先完成。

### 需要完成的前置工作
- 定义"四幕结构组装规则"（铺设 → 发展 → 高潮 → 收束）
- 建立"桥段幕位匹配表"（哪些桥段适合哪个幕位）
- 设计组装时的冲突处理（如多个母题叠加时优先级）

### 启用后效果
```
策划："帮我设计一个带水域BOSS的支线任务"
→ AI 自动完成：
  第一幕：选一个A悬念铺设桥段
  第二幕：选一个C恐怖或D战斗桥段
  第三幕：选一个E情感冲击或D战斗桥段
  第四幕：选一个J余韵留白桥段
→ 组装成完整任务骨架
```

### 重要边界
**即使启用，组装结果也是半成品**。策划仍需基于项目语境做二次创作。

## v3.4：作品评估系统

### 规划的评估维度
- **文学品质**（5 分）：结构完整度 / 角色塑造 / 主题深度
- **游戏适用性**（5 分）：任务节奏 / 玩家选择空间 / 重玩价值
- **社会文化适配**（5 分）：文化还原度 / 母题共识度 / 敏感度

### 启用后效果
```
策划："评估一下我这个任务设计"
→ AI 按三维度给分 + 具体改进建议
→ 15 分总分 + 星级评价
```

## v3.5：人物库回归（非神明角色）

### 背景
v2.0 的「社会身份角色索引」（635 条）在 v3.0 中删除。

### v3.5 规划
以**轻量化**方式重建：
- 不再做 23 类细分，改为"10-15 个核心职能原型"（如 反派官员 / 落魄文人 / 江湖侠女 / 神秘商贩 / 隐世高人）
- 每个原型挂接多个典型故事案例
- 直接从全文层角色档案字段中抽取，不单独建表

### 预期入口
恢复"入口③ 按角色原型查"，但概念与 v2.0 不同。

## v3.6：社会背景回归

### 背景
v3.0 聚焦素材检索，暂时不做世界观/社会背景导览。

### v3.6 规划
以**通用化**方式重建：
- 不绑定特定朝代
- 改为"古代社会通用知识模块"（官制 / 科举 / 丧葬 / 节庆 / 商业 / 婚俗 …）
- 每模块关联相关的 RI 仪式索引和 PL 场所索引
- 服务于"世界观搭建"类需求

### 恢复的入口
入口⑨ 查社会背景（通用版）。
