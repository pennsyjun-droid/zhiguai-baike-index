# 志怪百科索引 · Zhiguai Baike Index

一份面向游戏策划的志怪文学素材检索与角色卡生成 Skill。
基于唐传奇 156 篇、酉阳杂俎 277 条原始素材，结构化为 8 大索引约 2,400+ 条，可按需检索并生成叙事/战斗向的角色卡。

## 能解决什么痛点

- **查不到"能直接用"的素材**：通用搜索返回的是原文，不是策划可用的结构化信息。本 Skill 每条素材都标注了精彩度、核心能力、配套资源、游戏设计建议。
- **找到素材后还要自己组装**：原文 → NPC 卡 / BOSS 卡的转换成本高。本 Skill 提供两种卡模板 + 五条硬约束 + 七项质检，可直接生成可交付物。
- **素材脱离原典导致脑补失控**：AI 容易补充原文没有的细节，使策划误信。本 Skill 强制忠实性约束，原文没写的字段留白，不编造。
- **跨索引关联需要反复查**：一只妖怪的配套法术 / 道具 / 场所散在各索引。本 Skill 支持跨索引关联检索，一次查一只，拿到一整套。
- **能力边界不清导致期待错位**：本 Skill 明确列出"不做的事"（故事组装 / 作品评分 / 情绪筛选等），避免半成品交付。

## 30 秒上手

**4 种提问套路 + 1 种生成请求**，覆盖 90% 的使用场景：

| 你想要的 | 提问示范 | 走哪个入口 |
|---------|---------|-----------|
| 查某类素材 | "给我所有⭐⭐⭐龙蛇类妖怪" | ① 按素材类型 |
| 找故事套路 | "有没有人龙跨界恋情的经典案例" | ② 按母题 |
| 找场景单元 | "来个悬念铺设型开场桥段" | ③ 按桥段功能 |
| 查神明资料 | "给我钟馗的完整资料" | ④ 查神明 |
| 生成卡 | "MO-010 白蛇精 生成妖怪角色卡" | 卡生成分支 |

说具体 ID 或名称 + "生成 XX 卡" 即可触发卡生成。

## 五种常见误区

| 误区 | 替代方案 |
|------|---------|
| "帮我设计一个完整支线任务" | 拆为多次检索，自行组装（一键组装能力规划在 v3.3） |
| "评分这个设计方案" | 本期不做评估（v3.4） |
| "让玩家细思极恐的桥段" | 本期桥段检索不按情绪筛选，只按功能类（情绪在 v3.1） |
| "查唐代科举制度" | 本期不做社会背景（v3.6） |
| "BOSS 卡给我数值" | 不给。只给定性机制，数值由战斗策划自定 |

## 反馈

发现以下情况请直接在 [Issues](../../issues) 反馈：
